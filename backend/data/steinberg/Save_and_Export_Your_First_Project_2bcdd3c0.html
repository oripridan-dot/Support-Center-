Save and Export Your First Project

In this tutorial, we’ll cover the essential steps for saving and exporting your project. You’ll learn how to save your project to ensure your work is preserved and how to export your project as a WAV or MP3 file for sharing and playback.

Learning Goals:

  • Learn how to save your project.
  • Learn how to export your project as a WAV or MP3.

Part 1: Saving Your Project

Saving your project is crucial to ensure your work is preserved and can be reopened later. Cubase offers various saving options to protect your progress.

Steps to Save Your Project:

Save for the First Time:

  • Go to File > Save As or press Ctrl+Shift+S (Cmd+Shift+S on Mac).
  • In the dialog box, navigate to the folder where you want to save your project.
  • Enter a name for your project and click Save.

Save Regularly:

  • Once your project has been saved for the first time, you can simply press Ctrl+S (Cmd+S on Mac) or go to File > Save to save your progress as you work.

Save New Versions:

  • To create a backup or a new version of your project, use File > Save As and give the project a new name (e.g., "ProjectName_mix_final_v2").

Part 2: Exporting Your Project as a WAV or MP3

Exporting your project allows you to create a final audio file that can be played back on any device or shared with others. Cubase supports exporting in various formats, including WAV and MP3.

Steps to Export Your Project:

Set the Locators:

  • The left and right locators in the ruler at the top of the Project window define the start and end points of the section you want to export. You can just drag the locators to the desired positions.

Open the Export Audio Mixdown Dialog:

  • Go to File > Export > Audio Mixdown to open the export dialog.

Configure Export Settings:

  • Channel Selection: Select the mix you want to export (e.g., Stereo Out).
  • File Format: Choose the desired file format (e.g., WAV or MP3).
  • WAV: High-quality, uncompressed audio format.
  • MP3: Compressed audio format, suitable for sharing and playback on various devices.

Set File Location and Name:

  • In the File Location section, choose the folder where you want to save the exported file. Cubase generates a Mixdown folder in the project folder per default. This way your mixdown will always be close to your project.
  • Enter a name for the exported file.

Additional Settings:

  • Sample Rate: Typically set to 44.1 kHz for most applications, or higher for professional audio projects.
  • Bit Depth: Typically set to 16-bit for CD quality, or 24-bit for higher quality.
  • MP3 Settings: If exporting as MP3, you can adjust the bitrate (e.g., 192 kbps, 320 kbps) to balance between file size and audio quality.

Export the File:

  • Click Export to start the export process. Cubase will render the project to the specified file format and save it to the chosen location.

Tips for Effective Saving and Exporting

Save Frequently: Get into the habit of saving your project regularly to avoid losing your work.

Use Save As for Versions: Create new versions of your project using Save As to track different stages of your work.

Check Locator Position: Ensure that the position of the right locator leaves enough space of reverb tails and doesn't cut the end of your song.

Listen to Exported Files: Always listen to your exported files to ensure they sound as expected before sharing or publishing them.

By mastering the process of saving and exporting your projects in Cubase, you'll ensure your work is preserved and ready to be shared or published. These essential skills will enhance your workflow and help you produce high-quality audio files. Happy producing!
