SpectraLayers 11 Release Notes

SpectraLayers 11.0.70

June 5, 2025. We’re happy to announce the availability of the SpectraLayers 11.0.70 maintenance update. This new version provides resolutions to issues in different areas of the application.

Stability and Performance

  • An issue that occurred when shifting the tempo start in Pro Tools with SpectraLayers in ARA mode has been resolved.
  • An issue that affected  loading a Samplitude X8 project with SpectraLayers in ARA mode has been resolved.
  • An issue where, in specific cases, a 15 Khz tone appeared after using Unmix Song  has been resolved.
  • An issue that occurred when loading audio files on macOS 11 has been resolved.

SpectraLayers 11.0.60

January 23, 2025. We’re happy to announce the availability of the SpectraLayers 11.0.60 maintenance update. This new version introduces several issue resolutions and improvements throughout the application.

Improvements and New Features

  • Keyboard shortcuts to change the tool brush shape and units have been added.
  • The Bridge plug-in for Pro Tools now supports handles.
  • Tooltips with the full file path are now shown when hovering over recent projects in the home screen.
  • View > Time Unit > Tempo and Time Signature dialogs have been improved.
  • The time that it takes to display a long transcript has been reduced.
  • An error message is now displayed when using the Unmix Transcription function with an unsupported CPU on Windows.

Stability and Performance

  • A memory leak when downmixing the channel view or when soloing a channel has been fixed.
  • An issue with mouse events and the mouse cursor with Pro Tools in ARA mode has been resolved (macOS only).
  • An issue with the Pro Tools Bridge plug-in preview has been resolved (macOS only).
  • The Transfer tool no longer causes the display to blink when one layer is soloed.
  • Using Project > Reformat now keeps all project settings.
  • Tooltips now always show up with Pro Tools in ARA mode (macOS only).
  • Unmix Multiple Voices no longer freezes during processing.
  • When working with SpectraLayers as an external editor for DaVinci Resolve or other host applications, all edits are now applied correctly (Windows only).
  • Voice DeNoise now uses the reduction ratio percentage correctly when used with Background: Music.
  • Edit > Delete Time/Insert Time/Crop Time shortcuts are now accessible in the preferences.
  • Layer > Ungroup now properly selects layers after ungrouping.

SpectraLayers 11.0.50

November 21, 2024. We’re happy to announce the availability of the SpectraLayers 11.0.50 maintenance update. This new version introduces several issue resolutions and improvements throughout the application.

Improvements and New Features

  • More mouse wheel shortcuts, such as inverted scroll axis, have been added.
  • New keyboard shortcuts have been added to directly register profiles for specific modules.
  • ARA plug-in binding and unbinding is now much faster on tracks with hundreds of audio clips.
  • SpectraLayers now shifts focus back to Pro Tools when updating the edited file with the AudioSuite bridge plug-in.
  • Timecode metadata is now supported and can be saved as BWF metadata.
  • The Ambience Match module has a new "Replace" mode.
  • There’s a new Device Preference option to prevent “solo channel” from playing back on all device channels.
  • There’s a new contextual action on project tabs to show the project file in the system file browser.
  • You can now save or update – or ignore save/update – all projects at once when closing SpectraLayers.

Stability and Performance

  • An issue that occurred when exporting projects with metadata containing spaces in formats other than .wav has been resolved.
  • Dragging a layer out of the Layers panel no longer duplicates the layer.
  • Mouse shortcuts preferences are now saved properly.
  • Trying to preview certain modules after applying them no longer renders the application unreliable.
  • Unloading specific VST 3 plug-ins no longer renders the application unreliable.
  • Multiple issues with history and copies when duplicating a project have been resolved.
  • An issue with “solo channel” applied to all device channels, including unassigned channels, has been resolved.
  • Recovering a project after the application has become unreliable now works as expected.
  • An issue with module shortcuts in ARA mode on macOS has been resolved.
  • The DeClick module no longer creates audio artifacts.
  • Warning messages issued by a module within a module chain can now be closed.

SpectraLayers 11.0.40

October 28, 2024. We’re happy to announce the availability of the SpectraLayers 11.0.40 maintenance update. This new version introduces many issue resolutions and improvements throughout the application. This is also the first version of SpectraLayers that supports the AAX format and can be loaded as an ARA plug-in in Pro Tools.

Improvements and New Features

  • Enhancements regarding accessibility and compatibility for keyboard navigation and screen readers have been implemented.
  • New keyboard shortcuts have been added to apply modules directly.
  • SpectraLayers 11.0.40 can now be used in ARA mode in Pro Tools (version 2024.10 required).
  • Additional bitrate options are available for exporting multichannel projects.
  • More customizable mouse shortcuts have been added.
  • The size of the Module Presets selector has been increased.

Stability and Performance

  • Undocked panels are now properly hidden when the ARA plug-in is hidden.
  • Previously saved projects are now properly displayed in standalone mode.
  • SpectraLayers now performs accurate delay compensation when loaded as an ARA extension in Cubase and Studio One.
  • When using the Playback tool with a limited frequency range, the playhead now properly mirrors what is being played.
  • Using the DeClick module no longer creates audio artifacts at the beginning of a selection.
  • Rendering or removing SpectraLayers as an ARA extension and then undoing it now works as expected.

SpectraLayers 11.0.30

October 2, 2024. We’re happy to announce the availability of the SpectraLayers 11.0.30 maintenance update. This new version introduces many issue resolutions and improvements throughout the application.

Improvements and New Features

  • 3D channel configurations, such as Dolby Atmos and Ambisonics, with up to 16 channels are now supported.
  • When exporting only one layer, the name of the file is automatically assigned to the layer.
  • Selection events can now be excluded from the editing history.
  • The efficiency of the DeClick module has been improved.
  • The ARA display range has been improved, so that the selected clip is always visible.
  • Mouse wheel and touchpad navigation have been improved.

Stability and Performance

  • An audio dropout issue when switching between projects has been resolved.
  • The waveform power scrollbar now matches the default scale of +3 dB.
  • An issue when creating display and module filters has been resolved.
  • Multiple issues with the VST 3 effects module have been resolved.
  • An issue when loading audio files on macOS 11 has been resolved.
  • Enabling both Composite Layers view and the layer envelope no longer renders the application unreliable.
  • An issue when zooming a selected range has been resolved.
  • An issue when rendering waveforms without smoothing has been resolved.
  • Module presets are now properly saved in ARA mode.
  • A sync issue when exporting transcriptions as CSV files has been resolved.

SpectraLayers 11.0.20

August 21, 2024. We’re happy to announce the availability of the SpectraLayers 11.0.20 maintenance update. This new version introduces many issue resolutions  and improvements throughout the application.

Improvements and New Features

  • The Transient selection tool now features a Tolerance setting.
  • The quality of Unmix Song in Extreme mode has been improved.
  • OpenGL drivers are no longer required. Now only DirectX and Metal are used.
  • A shortcut to Redo (‘Ctrl+Shift+Z’) has been added for Windows.
  • Double-clicking a parameter while pressing the Alt/Option key now resets it to the default value.

Performance

  • The project size when performing local edits in SpectraLayers as an ARA extension has been optimized.
  • Various selection issues have been resolved.
  • Various issues with specific ARA hosts have been resolved.

Miscellaneous

  • The SpectraLayers AAX plug-in now opens in the foreground when called from Pro Tools.
  • Unmix Transcription now works as expected when selecting a language other than English.
  • An issue with the tone graph preview of the Signal Generator has been resolved.
  • An issue when resampling projects with extreme ratios, e.g. from 48Khz to 1Khz, has been resolved.
  • Overwriting a Display or Modules panel filter now works as expected.
  • A non-applied “preview” process is not visible anymore when switching tools via a keyboard shortcut.

SpectraLayers 11.0.10

July 1, 2024. We’re happy to announce the availability of the SpectraLayers 11.0.10 maintenance update. This new version introduces more than 20 urgently requested fixes and improvements, including full GPU acceleration on Windows, improvements related to Unmixing Performance and Rendering,as well as several fixes related to processing and editing. An important addition is the introduction of Batch Processing features also in ARA mode.

Improvements and New Features

Based on popular request, the Batch Processing has been added to the ARA mode as well (found in the Clip menu) and now features a new Export mode:
„Project“ – to save the result as a SpectraLayers project. The Polygonal Lasso now also gets dynamic fading options when right clicking during shape creationand the area to resize panels is now highlighted when hovered with the mouse. Several Layer Panel tooltips have been improved as well.

Performance improvements

  • Unmix Song is now fully GPU accelerated on Windows, allowing for a significantly better performance.
  • Unmix Song no longer hangs at the end of the unmixing process.

Modules and Processing

  • Drag and drop files in the Batch Processing dialog will now process files on Windows as expected.
  • DeReverb and DeClip modules no longer generate unwanted artefacts at regular intervals.
  • VST3 plug-in crashes have been fixed.
  • Using the Transfer tool in Composite mode with the target layer muted no longer results in a rending issue.
  • A graph rendering issue with EQ Match, Ambience Match, Reverb Match and DeNoise modules has been fixed.
  • Several issues related to applying or processing selections with fading on Windows have been fixed.

Miscellaneous

  • A warning message is now displayed on configurations where OpenGL 3.3 is not supported.
  • The Transfer tool now properly keep the chosen Target layer, and by default suggest layers within the current group.
  • Unmix Levels preview in loop mode now work as expected.
  • The cache folder would no longer fill up very quickly, in particular when using the Transfer tool.
  • The Pro Tools plugin on macOS no longer opens multiple SpectraLayers sessions instead of one.
  • Select > Load Selection now works properly in ARA mode.
  • The compact button in the tools area is now enabled in ARA mode.
  • Some issues related to certain audio devices have been fixed.
  • The VST3 Path Browse button is now properly shown the selected VST3 folder.
  • Some German translation typos have been fixed.
