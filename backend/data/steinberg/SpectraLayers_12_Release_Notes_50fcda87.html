SpectraLayers 12 Release Notes

SpectraLayers 12.0.40 Release Notes

November 26, 2025. We’re happy to announce the availability of a new SpectraLayers 12 maintenance update, version 12.0.40. This new version offers many issue resolutions and improvements throughout the application.

Improvements

  • GPU and RAM management has been optimized.

Miscellaneous

  • Pressing the ESC key now properly cancels a running process.
  • Moving a layer in timeline with the Transform tool no longer creates additional silence in some cases.
  • The preference “Display > Selected Layers” now works as expected.
  • Voice DeClip no longer generates low frequency audio artifacts.

SpectraLayers 12.0.30

September 25, 2025. We’re happy to announce the availability of a new SpectraLayers 12 maintenance update, version 12.0.30. This new version offers many issue resolutions and improvements throughout the application.

Improvements

  • The Voice Enhance module now features a new DeNoise First parameter that increases the enhancement quality.
  • Module positions now remain fixed, even after closing a session.
  • The Harmonic Selection tool has been optimized to significantly reduce slowdowns.

Tools and Modules

  • Using Unmix Soundtrack on mono projects no longer causes an increase in gain.
  • Occasional audio artifacts that were created when using Unmix Drums do not occur anymore.
  • Occasional audio artifacts that were created with the Guitar, Piano and Other layers when using Unmix Song do not occur anymore.
  • The Enhancement Ratio parameter in the Voice Enhance module now works as expected.

Stability

  • A memory leak issue that caused slowdowns when using the Clone Stamp tool has been resolved.
  • Stability issues when using the Signal Generator module with the Frequency Range Selection tool have been resolved.
  • Running SpectraLayers in the background no longer causes module processing to slow down (macOS only).
  • Using Unmix Song with limited VRAM no longer causes stability issues (Windows only).

Miscellaneous

  • Switching between different project tabs now properly reactivates the undo and redo buttons.
  • A blank dialog that used to automatically open when applying a module no longer appears (Windows only).
  • The SpectraLayers 12 ARA plug-in is now installed correctly after updating from a previous SpectraLayers 11 installation (macOS only).

SpectraLayers 12.0.20

August 27, 2025. We’re happy to announce the availability of a new SpectraLayers 12 maintenance update, version 12.0.20. This new version offers many issue resolutions and improvements throughout the application, including GPU acceleration on Windows for many Unmixing processes.

Improvements

  • The Unmix Soundtrack module is now GPU-accelerated on Windows.
  • The Unmix Noisy Speech and Voice Denoise options now detect vocal content above 12Khz, up to 16Khz or more, depending on the recording.
  • The Unmix Song module is now GPU-accelerated on Windows.
  • The "Unmix Sax & Brass First" option in the Unmix Song module is now properly saved in presets and restored in subsequent sessions.

Tools and Modules

  • The polygonal lasso tool now works as expected.
  • An issue with the time range selection tool not displaying selections properly when MIDI-log scale is used has been resolved.
  • The clone stamp tool now replicates data correctly.
  • The DeClip module no longer generates audio artifacts at regular intervals.

Stability

  • Applying a module with a keyboard shortcut without displaying its dialog no longer renders the application unreliable.
  • Applying modules with the standalone application in the background no longer leads to slower processing.

Miscellaneous

  • An issue where it was not possible to assign more than 127 audio device channels has been resolved.
  • Applying a module with specific audio channels selected on the Channels panel no longer creates silence in the unselected channels.
  • The Mute and Solo buttons now always react as expected on the Layers panel.
  • A playback synchronization issue that occurred when previewing a module in Loop mode has been resolved.

SpectraLayers 12.0.10

July 23, 2025. We’re happy to announce the availability of the first SpectraLayers 12 maintenance update, version 12.0.10. This new version offers many issue resolutions and improvements throughout the application. In addition to this, various user-requested new features have been implemented.

Please note that full GPU acceleration is not yet supported for Unmix Soundtrack and Unmix Song but will come with a later patch.

New Features and Improvements

  • The Unmix Drums module now encompasses a seventh kit-piece, including “Clap” extraction.
  • The buffer size of the VST 3 Effects module has been reduced, allowing for shorter latency and a more responsive module.
  • The quality of the Unmix Drums module has been improved, resulting in fewer artifacts for some elements (such as kick, snare, toms) and a better overall sound, which is closer to the original mix.
  • The responsiveness of the Harmonics Selection tool has been improved, allowing for faster and smoother selections.
  • SpectraLayers Elements 12 now also includes the Phase Rotation module.

Modules

  • An issue where audio artifacts were created when using Unmix Soundtrack with GPU acceleration on Windows has been resolved.
  • An issue where no audio was generated after using Unmix Song and Voice DeNoise in Music mode with AMD GPU on Windows has been resolved.
  • An issue where the Voice DeNoise module did not properly apply the reduction ratio has now been resolved.
  • The DeBleed module no longer generates unwanted audio artifacts.
  • Issues concerning several Unmix modules, such as processes being applied beyond the selected range, have been resolved.
  • An issue with the Signal Generator module, where no data was generated when using Insert mode at the beginning or end of a layer, has been resolved.

ARA

  • The cache folder now gets cleaned up properly after closing a project in ARA mode.
  • Applying a module in ARA mode on Windows while playing back the audio in the DAW no longer causes the application to close unexpectedly.
  • The Waveform Display Size and Layout Preset parameters are now working properly in ARA mode.

Miscellaneous

  • An issue where some VST 3 plug-ins were not displayed correctly has been resolved.
  • An issue where Select Harmonics generated inaccurate selections with the rectangular selection tool has been resolved.
  • An issue where the Clone Stamp tool added further data to existing data instead of replacing the data has been resolved.
  • An issue with transport and the view resetting to the stop position when applying a module has been resolved.
