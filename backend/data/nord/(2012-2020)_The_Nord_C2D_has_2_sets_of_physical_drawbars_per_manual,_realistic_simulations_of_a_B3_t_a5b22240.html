Nord C2D

(2012-2020) The Nord C2D has 2 sets of physical drawbars per manual, realistic simulations of a B3 tone wheel organ, Vox and Farfisa transistor organs and a beautiful sampled baroque pipe organ the new Nord C2D is the ultimate portable organ solution.

Clavia USB Driver

Download for Windows

Clavia USB Driver Setup.zip

The installer will run on 64-bit versions of Windows 7, 8, 10, 11 and will by default install both the legacy v3.x driver as well as the v4.x Nord Keyboards Driver, which is required for compatibility with later Nord instruments. The installer also checks for currently installed versions of Nord Sound Manager and Nord Sample Editor and will inform if any of these applications need to be updated for compatibility with the v4.x Nord Keyboards Driver.
